# Web App (Angular 20 + Tailwind + PrimeNG)

This is a full Angular app scaffold ready for Vercel deployment.

## Features included

- Angular 20 standalone architecture
- Tailwind CSS + PrimeNG UI wiring
- JWT auth service + interceptor + route guard
- 15-day frontend session expiry policy
- Feature pages: Dashboard, Services, Songs, Schedules
- PWA basics (`manifest.webmanifest` + `ngsw-config.json`)

## Run locally

```bash
npm install
npm run start
```

## Build

```bash
npm run build
```
